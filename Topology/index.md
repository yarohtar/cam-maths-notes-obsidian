[[Topological space]]
Some topologies are induced by a [[Metric Space|metric]].
[[Hausdorff]]
$A\subset X$ is open if every point has an open [[Neighbourhood|nbd]] that is contained in $A$
$x_n\to x$ if for any nbd $V$ of $x$ we find $N$ s.t. $x_n\in V$ for $n>N$
In a [[Hausdorff]], the limits are unique.
Suppose $A\subset X$ is closed and $x_n\to x$ in $X$ and $x\not\in A$. 
Then no open nbd of $x$ is in $X\setminus A$ 
but then $X\setminus A$ is not open contradiction so it has to be $x\in A$.
HOWEVER, a set $A$ which contains all its limit points no longer has to be closed.
