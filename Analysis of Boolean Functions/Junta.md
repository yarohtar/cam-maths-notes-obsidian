Let $f:\{ -1,1 \}^{n}\to \{ -1,1 \}$. 
Suppose that there is some $J\subseteq[n]$ such that $f(x)=f(y)$ whenever $x|_{J}=y|_{J}$
(i.e. $f$ only depends on coordinates from $J$).
Then $f$ is called a $J$-junta. 
If there is some $J$ of size $k$ such that $f$ is a $J$-junta then $f$ is a $k$-junta.
The set $J$ itself is also sometimes called a junta.
