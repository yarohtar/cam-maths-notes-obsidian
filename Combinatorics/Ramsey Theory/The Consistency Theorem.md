**Corollary 19 (The Consistency Theorem).** If $A$ and $B$ are partition regular then the matrix $\begin{pmatrix} 0 & A \\ B & 0 \end{pmatrix}$ is also partition regular. In other words, if we can guarantee to solve $Ax = 0$ in some colour class and $By = 0$ in some colour class then we can guarantee to solve both in the same colour class.

**Remark.** This is not obvious by considerations of partition regularity alone.
