Let $X\subseteq \mathbb{N}$ be finite.
We say that $L\subseteq X^{n}$ is a combinatorial line 
if there are some active coordinates $i_{1},i_{2},\dots,i_{r}\in[n]$,
with $0<r<n$ and some $a_{i}\in X$ such that 
$$
L=\{ x\in X^{n} : x_{i}=a_{i}\text{ for $i\not\in I$ and }x_{i_{1}}=x_{i_{2}}=\dots=x_{i_{r}} \}
$$
with the set $I=\{ i_{j}:j\in [r] \}$.
