Let $m,k,d\in \mathbb{N}$.
Then there is some $n\in \mathbb{N}$ 
such that whenever $[m]^{n}$ is $k$-coloured,
there exists a monochromatic $d$-[[Parameter Set]].
### Proof
Let $n'=HJ(m^{d},k)$.
By [[The Hales-Jewett Theorem]], find a [[Combinatorial Line]] in $[m^{d}]^{n'}$.
This is now a $d$-[[Parameter Set]] in $[m]^{dn'}$, so set $n=dHJ(m^{d},k)$.
