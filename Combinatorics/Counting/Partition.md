A partition of a set $A$ is a set $\pi$ such that 
$$
\begin{gather}
A = \bigcup_{x\in \pi} x\\
x,y\in \pi \implies x\cap y=\varnothing \lor x=y
\end{gather}
$$
Given a finite set $A$ we define a [[Combinatorial Structure]] 
$$
P(A) = \{ \{ S_{1},S_{2},\dots,S_{k} \} : S_{1}\sqcup\dots \sqcup S_{k}=A;\, S_{i}\neq \varnothing \}
$$
the set of partitions of $A$ into nonempty parts.
