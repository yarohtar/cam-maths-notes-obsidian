In [[Zermelo-Fraenkel Set Theory]]
For a set $x$, define the successor of $x$ to be the set $x^{+}=x\cup \{ x \}$
We introduce some names for these:
$0=\emptyset$
$1=\emptyset^{+}=\{ 0 \}$ 
$2=1^{+}=\{ 0,1 \}$
$3=2^{+}=\{ 0,1,2 \}$
...

### Definition
Say '$x$ is a successor set' if 
$(0\in x)\land(\forall y\in x)(y^{+}\in x)$
where $(\forall)$
