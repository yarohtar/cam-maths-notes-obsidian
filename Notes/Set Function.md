Let $\mathcal{A}$ be any set of subsets of $E$ containing $\emptyset$.
A set function is a function $\mu:\mathcal{A}\to[0,\infty]$ with $\mu(\emptyset)=0$.

### Properties
- $\mu$ is increasing if for all $A,B\in \mathcal{A}$ with $A\subseteq B$
$$
\mu(A)\leq \mu(B)
