We say $(P,\leq)$ is a poset if $\leq$ is a [[Partial Order]] on $P$.
The [[Topological space|Topology]] on $P$ is given by base open sets of the form
$$
\{ q \in P: q\leq p \}
$$
for some $p\in P$. 
In particular a set in $P$ is open if and only if it is downwards closed.
