Let $X$ be a [[Topological space]]. 
We define the boldface Borel hierarchy, consisting of classes $\mathbb{\Sigma}_{\alpha}^{0}$, $\mathbb{\Pi}_{\alpha}^{0}$ and $\mathbb{\Delta}_{\alpha}^{0}$.
### $\mathbb{\Sigma}$
A set $A\subseteq X$ is $\mathbb{\Sigma}_{1}^{0}$ if it is open
A set $A\subseteq X$ is $\mathbb{\Sigma}_{\alpha}^{0}$ if there are sets $A_{1},A_{2},\dots$ such that 
$$
A=\bigcup_{i\in \omega} A_{i}
$$
and $A_{i}$ is $\mathbb{\Pi}_{\alpha_{i}}^{0}$ for some $\alpha_{i}<\alpha$. 
### $\mathbb{\Pi}$
A set $A\subseteq X$ is $\mathbb{\Pi}_{\alpha}^{0}$ if $X\setminus A$ is $\mathbb{\Sigma}_{\alpha}^{0}$
### $\mathbb{\Delta}$
A set $A\subseteq X$ is $\mathbb{\Delta}_{\alpha}^{0}$ if it is both $\mathbb{\Sigma}_{\alpha}^{0}$ and $\mathbb{\Pi}_{\alpha}^{0}$.
